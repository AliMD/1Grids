Download and use

Public cdn.

<link href="//cdn.rawgit.com/AliMD/1Grids/v4.0.0/1grids.css" type="text/css" rel="stylesheet" />

Bower

bower install 1Grids

NPM

npm install 1Grids

Sass

import 1Grids folder in your sass project

// grid_num = 12
@import '1Grids/'
